# 📊 Dashboard de Gestión Operativa y Capital Humano

Este proyecto presenta una solución integral para el control de operaciones y gestión de personal, desarrollada por una **Ingeniera Industrial** con enfoque en eficiencia y cumplimiento.

### 🛠️ Funcionalidades Clave:
* **Control de Asistencia:** Seguimiento detallado de incidencias, vacaciones y presencia.
* **KPIs Estratégicos:** Monitoreo de Índice de Ausentismo (Meta vs Real), Horas Hombre Disponibles y Eficiencia de Personal.
* **Gestión de Incidencias:** Base de datos estructurada para el registro de bajas y permisos.

### 🚀 Impacto:
Permite una visibilidad clara de la **Disponibilidad Operativa**, facilitando la toma de decisiones para reducir costos por ausentismo y optimizar la productividad del equipo.
